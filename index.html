Skills

I have basic experience in a pretty wide array of languages, however in only a few of these could I work on a large project without external help or reference.

The languages i am best are (in order of experience) are as follows:

1. Node JS
this is the language which i have used the most, from building a social media from scratch to real time kahoot clone, Node has always been my go to tool for an easy and reliable language.
2. Vanilla JS
I love using vanilla js to build random and silly projects like small games quickly, vanilla js was what got me into programming in the first place.
3. Unity C#
I discovered that I enjoyed making games with vanilla js, so i went to Unity for a game engine that's stood the test of time and cemented it self as one of the best. I am currently using unity to develop my RPG game, Sedimentary.
4. C++
I am still learning, however using C++ is always enjoyable, going down close to hardware feels so good and "clean", I am currently using C++ to develop my CLI and my very own programming language
